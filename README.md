Simple Template Layouts
====

### Init Project

	# Clone https://github.com/ctekjlo/Layout
	git clone https://github.com/ctekjlo/Layout.git project_dir

	# Go to project dir
	cd project_dir

	# Install bower components http://bower.io/
	bower install

	# Go to production dir
	cd production

	# Run localhost server
	server
